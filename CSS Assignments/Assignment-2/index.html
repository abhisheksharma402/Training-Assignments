1. Size Units (px, %, vw, vh, em, rem)

Pixels specify definite or absolute values. They are exact values and not relative to anything. They will always carry that exact value always, at every screen size.

Percentages(%) specify values relative to the parent element. In this example, parent's width is 50% relative to its parent (outer div with class - "size-units" that has 100% width of body (its parent)) and child's width is 50% relative to its parent (div with class "parent"). Same is case with font-size. Default font-size of html element or root element is 16px. Usually child's font size is 100% of its parent. But, here the parent element has font size equivalent to 100% of its parent. So, this text has 16px.

vw (viewport-width) is relative to the browser's default viewport size. Its very much similar to percentages but the only difference is that %ges also take margins, and padding into account while calculating the width of the element while vw doesn't. vh is same as vw but for height. This elements has height = 30vh.

rem is relative to the font-size of the root element or the HTML element. Font size of this text is 200% of 16px which is 32px.

This is inside em. Font size of this text is 2*(font-size of HTML) = 2*16 = 32px

em is relative to the font-size of the parent element. Font size of this text is 1*(font-size of div with class "em") = 1*32 = 32px.

4. CSS Combinators

A combinator is something that explains the relationship between the selectors.

4.1. Descendant Selector

The descendant selector selects all the descendants (child, grandchild and so on) of the specified element.

Hi, I am des

Hello there, I'm the child of des

Hey, I am the grandchild of des

4.2. Child Selector (>)

The child selector selects all elements that are the children of a specified element.

Hi, I am des

Hello there, I'm the first child of des

Hey, I am the grandchild of des

Hello, I am the second child of des

4.3. Adjacent Sibling Selector (+)

The adjacent sibling selector is used to select an element that is directly after another specific element.

Hi, I am des

Hello there, I'm the first child of des

Hey, I am adjacent to the above paragraph tag

Hey, I am the grandchild of des

Hello, I am the second child of des and a sibling of first paragraph tag, but not adjacent

4.4. General Sibling Selector (~)

The general sibling selector selects all elements that are next siblings of a specified element.

Hi, I am des

Hello there, I'm the first child of des

Hey, I am adjacent sibling of the above paragraph tag

Hey, I am the grandchild of des

Hello, I am the second child of des and a sibling of first paragraph tag
